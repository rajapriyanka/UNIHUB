## âš™ï¸ Running the Project

### 1. Clone the Repository

```bash
git clone https://github.com/rajapriyanka/UNIHUB.git
cd UNIHUB
```

### 2. Run Backend (Spring Boot)

```bash
cd "Spring boot"
./mvnw spring-boot:run
```

> Default port: `http://localhost:8080`

```bash
cd ../Frontend

```
